In Quantum Espresso, phonon dispersion is computed using the `ph.x` program, which is based on Density Functional Perturbation Theory (DFPT).

## Theory

Consider a unit cell containing $N$ atoms.

$s=1…N$ : Index of atoms in the unit cell

$\alpha,\beta = x,y,z$ : Indices for Cartesian coordinates

$\boldsymbol R$ : Position vector of the unit cell in the Bravais lattice

$\boldsymbol u_{s\alpha}(\boldsymbol R)$ : Displacement component of the $s$-th atom in the $\alpha$ direction

Real-space interatomic force constants:

$$
\Phi_{s\alpha,s'\beta}(\boldsymbol R,\boldsymbol R')=\frac{\partial^2 E_\text{tot}}{\partial \boldsymbol u_{s\alpha}(\boldsymbol R)\partial \boldsymbol u_{s'\beta}(\boldsymbol R')}
$$

$\boldsymbol q$-space interatomic force constants:

$$
\widetilde\Phi_{s\alpha,s'\beta}(\boldsymbol q,\boldsymbol q')=\frac{\partial^2 E_\text{tot}}{\partial \boldsymbol u_{s\alpha}(\boldsymbol q)\partial \boldsymbol u_{s'\beta}(\boldsymbol q')}
$$

Fourier transformation:

$$
\widetilde {\boldsymbol u}*{s\alpha}(\boldsymbol q) = \sum*{\boldsymbol R}\boldsymbol u_{s\alpha}(\boldsymbol R) e^{-i\boldsymbol q\cdot \boldsymbol R}
$$

According to Bloch's theorem:

$$
\frac{\partial^2 E_\text{tot}}{\partial \boldsymbol u_{s\alpha}(\boldsymbol q)\partial \boldsymbol u_{s'\beta}(\boldsymbol q')} = \delta_{\boldsymbol q,\boldsymbol q'}\frac{\partial^2 E_\text{tot}}{\partial \boldsymbol u_{s\alpha}(\boldsymbol q)\partial \boldsymbol u_{s'\beta}(\boldsymbol q)}
$$

The interatomic force constants $\widetilde\Phi_{s\alpha,s'\beta}(\boldsymbol q,\boldsymbol q')$ in $\boldsymbol q$-space are easier to compute, and then the real-space interatomic force constants $\Phi_{s\alpha,s'\beta}(\boldsymbol R,\boldsymbol R')$ are obtained through inverse Fourier transformation.

The frequencies and eigenvectors of the normal modes are determined by the secular equation:

$$
\sum_{s',\beta}\widetilde {\boldsymbol D}*{s\alpha,s'\beta}(\boldsymbol q)\widetilde {\boldsymbol u}*{s'\beta}(\boldsymbol q) = \omega_{\boldsymbol q}^2\widetilde {\boldsymbol u}_{s\alpha}(\boldsymbol q)
$$

where the dynamical matrix is:

$$
\widetilde {\boldsymbol D}*{s\alpha,s'\beta}(\boldsymbol q) = \frac{1}{\sqrt{M_sM*{s'}}}\sum_{\boldsymbol R,\boldsymbol R'}\Phi_{s\alpha,s'\beta}(\boldsymbol R,\boldsymbol R') e^{i\boldsymbol q(\boldsymbol R'-\boldsymbol R)}
$$

### (1) Perform SCF calculation to obtain the self-consistent electronic ground state energy using `pw.x`.

- **SCF input file (`scf.graphene.in`)**

```bash
&control
  calculation = 'scf'
  restart_mode = 'from_scratch'
  pseudo_dir = './pseudos/'
  outdir = './out/'
  prefix = 'graphene'
/
&system
  ibrav = 4
  celldm(1) = 4.654
  celldm(3) = 6.0
  nat = 2
  ntyp = 1
  ecutwfc = 60.0
  ecutrho = 600.0
  assume_isolated = '2D'
/
&electrons
  conv_thr = 1.0e-8
/
ATOMIC_SPECIES
  C 12.0107 C.pbe-n-kjpaw_psl.1.0.0.UPF
ATOMIC_POSITIONS alat
  C 0.00000 0.00000 0.00000
  C 0.00000 0.57735 0.00000
K_POINTS {automatic}
  8 8 1 0 0 0

```
